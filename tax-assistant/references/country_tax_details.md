# Country Tax Details (Stub)

> Replace this stub with authoritative information for the country whose tax regime you are processing.

## Required Sections

1. **Jurisdiction Overview**
   - Tax authority name and primary reference links.
   - Official fiscal year start/end dates.
   - Local currency and rounding/precision expectations.

2. **Global Adjustments**
   - Required exchange-rate sources and frequency (e.g., monthly averages, spot rates).
   - Any compulsory inflation, interest, or withholding adjustments.

3. **Reporting Fields**
   - Enumerate the standard tax codes or spreadsheet tabs that every institution should map to.
   - Note any mandatory supporting schedules or disclosures.

4. **Asset & Liability Guidance**
   - Summarize classification rules for assets, liabilities, and capital gains.
   - Include references to any statutory guides or publications.

5. **Audit & Compliance**
   - Outline documentation retention requirements.
   - List country-specific sign-off or attestation steps.

Populate each section with concise, actionable guidance before processing client data. Agents should refresh this file whenever the jurisdiction changes or regulations are updated.
